# DiscordBotPricillaPearson
This is a test project to create Discord Bot for ICU (Interstellar Communist Union) - community of Russian-speaking
players in the game Elite Dangerous. 

The project based on JDA (Java Discord API) and JDBC.

# Commands
 - !coriolis - link to Elite: Dangerous Shipyard
 - !eddb - link to Elite: Dangerous Database
 - !lastexp - last expansion of Nagii Union faction
 - !expdate [system name] - date of expansion into the 'system name'
 - !syscount - number of systems Nagii Union fraction
 - !forum - link to forum ICU
 - !inara - link to Elite: Dangerous companion
 - !online - channel activity information
